# REV Opacity Cutout Documentation

Welcome to the documentation for the REVO Opacity Cutout Blender add-on.

REVO Opacity Cutout converts opacity-mask textures into clean cutout geometry directly in Blender. It is built for foliage-heavy Unreal Engine workflows, including Nanite-ready cards and Megascans-style vertex color wind setup.

## Local Development

1. Install dependencies.
2. Run `mkdocs serve`.
3. Open `http://127.0.0.1:8000/`.
4. Open Getting Started at `http://127.0.0.1:8000/getting-started/`.

Source-of-truth note:

- Write docs only in files under `docs/`.
- Do not edit files in `site/` (generated output).

## Build

Run `mkdocs build` to generate the static site in the `site` folder.
